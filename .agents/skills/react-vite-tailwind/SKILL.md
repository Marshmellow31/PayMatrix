# React Vite Tailwind PWA Skill

## Requirements based on README
- **React (18+)**: UI Library for the modern web app.
- **Tailwind CSS (3+)**: Utility-First CSS Framework for styling.
- **Vite (5+)**: Fast build tool & dev server.
- **Redux Toolkit (2+)**: State Management.
- **React Router (6+)**: Client-Side Routing.
- **Framer Motion & Chart.js**: Animations and visual charts.

## Configuration & Usage Rules
- All frontend code resides in the `frontend` directory.
- Use `npm run dev` to start the frontend server on port 5173.
- Use functional components with hooks.
- Use Tailwind for styling. No vanilla CSS outside index.css.
- Redux slice patterns are to be placed in `frontend/src/redux/`.
- The PWA manifest and service worker config must maintain a Lighthouse target score of 90+.
